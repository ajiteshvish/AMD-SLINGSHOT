# AMD Technology Stack

## Purpose

URTSS leverages AMD’s AI software ecosystem to deliver fast, efficient, and privacy-first AI inference on consumer devices.

---

## Technologies Used

### ROCm

- Enables GPU-accelerated AI inference
- Used for high-performance trust scoring and NLP tasks

### ONNX Runtime (AMD Support)

- Executes AI models locally
- Ensures cross-device compatibility
- Eliminates cloud dependency

### INT8 Quantization

- Reduces model size and latency
- Improves performance and power efficiency

### Ryzen AI Software

- Enables AI execution on AMD Ryzen laptops and PCs
- Optimized for consumer devices

### ZenDNN

- Accelerates AI inference on AMD CPUs
- Ensures performance even without GPU usage

### Vitis AI

- Used during model development and optimization
- Simplifies testing and deployment

---

## Value of AMD Stack

- Faster inference
- Lower power consumption
- Enhanced privacy
- Scalable from MVP to enterprise
