# MVP Scope

## Included in MVP

- Unified retailer trust score
- Explainable score breakdown
- AI-based review sentiment analysis
- Fake review detection
- Seller comparison view
- Trust badges
- Local AI inference

---

## Excluded from MVP

- Cross-marketplace identity
- Image authenticity checks
- Voice-based queries
- Enterprise dashboards

---

## MVP Goal

Demonstrate a working AI-powered trust scoring pipeline with real consumer impact.
