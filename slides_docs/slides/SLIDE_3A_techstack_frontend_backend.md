# Tech Stack — Frontend & Backend

## Frontend

### Core

- React 19 + TypeScript
- Vite 7 (SWC plugin)
- React Router DOM v7

### Styling & Animation

- Tailwind CSS v3
- Framer Motion v12
- tsParticles (hero background)
- Lucide React (icons)

### Data & Charts

- Recharts v3 (trust score graphs)
- Axios (API calls)
- react-countup (animated numbers)

### Auth & DB (Client)

- Firebase SDK (Auth: Google + Email)
- Supabase JS v2 (direct DB queries)

## Backend

### Framework & Server

- FastAPI (Python REST API)
- Uvicorn (ASGI server)
- Python 3.11

### Validation & Config

- Pydantic + Pydantic Settings
- python-dotenv (.env management)

### Database Client

- Supabase Python SDK
- PostgREST (auto REST from Postgres)
