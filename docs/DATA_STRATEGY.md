# Data Strategy

## MVP Data Sources

- Synthetic retailer datasets
- Publicly available review samples
- Simulated delivery and return metrics

---

## Data Fields Used

- Seller ID
- Review text
- Star ratings
- Delivery success rate
- Return/refund rate
- Customer response behavior

---

## Privacy Approach

- No personal user data stored
- No private marketplace data accessed
- All AI processing occurs locally

---

## Future Data Integration

- Secure API-based data sharing
- Marketplace SDK integration
- Controlled access under agreements
